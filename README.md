MANUAL INSTALL

#BACKEND

1. cd backend
2. run 'npm install'
3. run 'npm run start'
(server will starting at http://localhost:3000)

#FRONTEND

1. cd frontend
2. run 'npm install'
3. run 'npm build'
4. run 'npm run start'
(site will starting at http://localhost:8080)


DOCKER INSTALL

1. run 'docker compose up'
(app will starting at http://localhost:8080)
